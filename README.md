# photo-legend
Projeto destinado a adicionar uma legenda na foto.

# Maven

```xml
<dependency>
     <groupId>br.com.involves</groupId>
     <artifactId>photo-legend</artifactId>
     <version>${photo-legend.version}</version>
 </dependency>
 ```


# Utilizar

```java
PhotoLegend.getInstance().addLegend("[LABEL]", [OBJECT]);
```
